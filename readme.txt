Plugin de Wordpress para el procesamiento de pagos con ENZONA con integración de WooCommerce.

== Installation ==

1. Descompactar el archivo .zip descargado.
2. Subir la carpeta del plugin en el directorio `wp-content/plugins/` del sitio con WordPress.
3. Activar `EnzonaPay` en la página de Plugins.
4. Llenar los datos solicitados para la configuración del plugin en la sección de pagos de WooCommerce.
